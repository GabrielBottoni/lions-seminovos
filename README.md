# Desafio Técnico: Desenvolvedor Front-end

Este projeto foi desenvolvido como parte do processo seletivo para a vaga de **Desenvolvedor(a) Frontend**. A aplicação consiste em uma SPA (Single Page Application) moderna, focada em performance e usabilidade.

---

## Desenvolvimento do Layout

### Conceito

- O design foi projetado para transmitir modernidade e confiança. Sabendo que o público desse nicho busca objetividade, a interface foi estruturada para destacar os pontos de maior valor, utilizando uma hierarquia visual que direciona o olhar do usuário para as ações principais.

### Foco em Conversão (CTA)

- A estratégia de UX (User Experience) foi centrada em um CTA (Call to Action) eficiente. Os elementos de interação foram posicionados de forma estratégica para facilitar a jornada do cliente, garantindo que o botão de ação seja sempre visível e convidativo, maximizando as chances de conversão.

### Diferenciais:

- Navegação Fluida: Resposta imediata aos comandos, essencial para o público mobile.

- Design Responsivo: Interface que se adapta perfeitamente, de telas de smartphones a monitores de alta resolução.

- Estética Premium: Uso de componentes Bootstrap personalizados para uma aparência moderna e profissional.

## Tecnologias e Ferramentas

Optei pelas seguintes ferramentas para garantir um código limpo e escalável:

* **[React 19](https://react.dev/):** Versão mais recente da biblioteca para interfaces de usuário.
* **[Vite](https://vitejs.dev/):** Ferramenta de build e servidor de desenvolvimento de alta performance.
* **[Bootstrap 5](https://getbootstrap.com/) & [React-Bootstrap](https://react-bootstrap.github.io/):** Estilização e componentes de interface prontos para uso.
* **[React Router Dom](https://reactrouter.com/):** Gerenciamento de navegação e rotas dinâmicas.
* **[React Icons](https://react-icons.github.io/react-icons/):** Biblioteca para inclusão de ícones como componentes.
* **[ESLint](https://eslint.org/):** Padronização e qualidade de código.

---

## Organização do Projeto

A organização das pastas foi pensada para facilitar a manutenção:

* `src/main.jsx`: Ponto de entrada que configura o React e importa os estilos globais.
* `src/App.jsx`: Definição das rotas e estrutura principal do layout.
* `src/components/`: Componentes modulares e reutilizáveis da aplicação.
* `src/pages/`: Telas completas que representam as visualizações da rota.

##  Como Rodar o Projeto

Siga os passos abaixo para executar a aplicação localmente.

### 1. Clonar o Repositório

```bash
git clone [https://github.com/seu-usuario/nome-do-repositorio.git](https://github.com/seu-usuario/nome-do-repositorio.git)
cd nome-do-repositorio
```

### 2. Instalar Dependências

Baixe as bibliotecas necessárias listadas no package.json:
```bash
npm install
```

### 3. Rodar o Servidor de Desenvolvimento

Inicie o projeto no navegador com atualização automática:
```bash
npm run dev
```
Geralmente, o projeto ficará disponível no endereço: http://localhost:5173
